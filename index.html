TouchEn mVaccine — 모바일 보안 SDK 개발·운영

2019.04 — 현재
메인 담당 — Android/iOS SDK/App · 라온시큐어

200여 금융·공공기관에 납품하는 모바일 보안 SDK. 1,000만 사용자 대상 서비스 중. 진화하는 루팅/탈옥 기법에 대응하는 탐지 시스템을 운영하고, 공격 시뮬레이션으로 방어 로직을 검증하는 사이클을 돌려왔다.

루팅 탐지 우회 벡터를 자체 공격 시뮬레이션으로 발견하고, NDK 레벨 방어 로직을 강화하는 탐지-검증 사이클을 운영. KISA 백신 보안 성능 인증 획득.
  • 탐지·방어: Magisk/RootHide/KernelSU 등 진화하는 루팅 기법 분석 및 탐지 구현. Frida Java/Native 후킹 기반 공격 시뮬레이션으로 탐지 우회 벡터를 발견하고 NDK 레벨 방어 로직 강화. 안티프리다 탐지 구현.
  • 공격 시뮬레이션: FakeCall, BankingTrojan 등 런타임 후킹 기반 공격 PoC를 작성하여 방어 로직 사전 검증. 코드 난독화 및 안티디버깅 로직으로 SDK 리버싱·변조 방지.
  • iOS 보안: 탈옥 탐지, 원격 탐지, 디버깅 탐지 모듈 개발. iOS 환경의 보안 위협에 대응하는 탐지 기능 구현.
  • 탐지→대응 흐름: 악성앱/보이스피싱앱 탐지 → 위험 앱 리스트 노출 → 사용자 삭제 선택까지 전체 흐름 구현.
  • 앱 간·앱-웹 통신: URL Scheme 기반 앱-웹 간 통신 및 AIDL/Bound Service 기반 앱 간 통신 기능 개발.
  • SDK 엔지니어링: Java→Kotlin 99% 전환과 멀티모듈 아키텍처 전환을 public API 하위 호환 유지하며 수행. ANR과 메모리 릭을 JNI 레벨까지 추적·해결.
  • SDK 운영: Android 8~15(7개 메이저 버전), 삼성·LG·샤오미·Pixel 호환성 검증. 분기 릴리즈 사이클로 200여 고객사 안정적 배포. 플레이스토어 배포 및 업데이트 운영. 행정안전부 폐쇄망 백신 앱 납품.

AI 딥페이크 탐지 — 온디바이스 vs 클라우드 아키텍처 전환 판단

2024.04 — 2024.12
앱 개발 리드 / 모델 개발 서브 (6인 팀) · 라온시큐어

오픈소스 딥페이크 탐지 모델을 모바일 보안 서비스에 응용. 모델 성능·추론 속도·UX 트레이드오프를 분석해 아키텍처 전환을 결정했다.

자체 데이터셋 구축으로 정확도 30%p 개선(95%). TFLite 양자화(FP32→FP16→INT8) 적용 후 온디바이스 추론 속도와 MediaProjection 해상도 제약(720p, 30fps 미달)을 측정 → 클라우드 비동기 아키텍처로 전환하여 서비스 출시 결정.
  • 자체 데이터셋을 구축하고, i3d·ResNet 기반 탐지 모델의 정확도를 65%→95%로 개선. 온디바이스 추론의 한계를 측정하여 클라우드로 전환.
  • 50만 다운로드 B2C 앱에 탑재하여 실서비스 출시. 사용자 영상의 딥페이크 여부를 확률(%)로 제공하는 탐지 기능 구현.

AI 보이스피싱 탐지 — FP Rate 분석 기반 제품화 판단

2025.09 — 2025.12
전체/모델 개발 리드 (3인 팀) · 라온시큐어

보이스피싱 통화를 AI로 실시간 탐지하는 연구. Recall 99.34%에도 FP Rate 22%로 실서비스 품질에 미달하여 제품화를 보류.

Recall 99.34%이나 FP Rate 22%(정상 통화 5건 중 1건 오탐) — 실사용하기엔 오탐이 너무 많아 출시 보류. 후속 개선 전략(Hard Negative Mining, 정상 대화 데이터 보강 및 합성 데이터 보강) 수립.
  • KoBERT 기반 보이스피싱 탐지 모델 설계 및 학습 파이프라인 구축. 후속 개선 방향으로 정상 대화 데이터 보강 및 Hard Negative Mining 전략 제안.
  • 모델 학습 파이프라인을 구축하고 추론 흐름을 구현. 모델 성능 지표뿐 아니라 체감 품질까지 검증하여 상용화 판단 결정.
